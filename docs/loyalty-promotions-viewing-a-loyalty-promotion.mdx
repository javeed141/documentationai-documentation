---
title: View a Loyalty Promotion
---

The Promotions dashboard is the hub for managing your loyalty promotions. From this page, you can quickly see each promotion's loyalty program, status (such as Live or Awaiting Approval), scheduled duration, and last modification date.

To manage your view, use the search bar to find a specific promotion or apply filters to narrow the list. Use the sorting options to organize promotions by criteria such as status or update date. To create a promotion quickly, use the duplicate action to copy an existing one.

<Callout kind="info" title="Note">

The list view only shows the promotions created on the loyalty promotions UI.

</Callout>

## Searching for a loyalty promotion

Use the search bar to find a specific promotion. You can search by description, promotion ID, or external identifier in addition to the promotion name.

To search for a loyalty promotion, use the search bar and provide a valid loyalty promotion name or external ID.

## Sorting and filtering loyalty promotions

Use sorting and filtering to organize and refine your view of loyalty promotions. You can use filtering to narrow the list and focus on a specific subset of promotions, for example, viewing only Live promotions for a particular program or all promotions updated by a specific team member.

To sort and filter loyalty promotions, follow these steps:

1. Select the filter icon on the listing page.
2. Select the loyalty program from the Select program dropdown list.
3. Select the status from the Select status dropdown list.
4. Select the start date and end date under Duration.
5. Select a user from the Select name dropdown list.
6. Select Apply to apply the filter.

<Image width="80%" src="https://files.readme.io/e1c6e70cb9bf6b4d40857091cd1fa3093b28459b792c19edc20d4c93fc5e5d36-image.png" />

After applying these actions, your promotions list will be refined, displaying only the promotions that match your search and filter criteria, making them easier to manage.

## Viewing analytics for a loyalty promotion

View the analytics report for a loyalty promotion and key metrics, such as member enrolment, opt-in rates, points issued, and redemption statistics, in a single, consolidated view.

To view the analytics for a loyalty promotion, follow these steps:

1. In the promotion list view, go to the loyalty promotion.
2. Hover over the More options (three vertical dots) menu, and then select View analytics.

<Image width="80%" src="https://files.readme.io/f73e7a4861415c4bbb9156fefd11ab9c414ba988d6c5fd5f3659eb0298d72bcc-image.png" />

3. Select a metric category to view the analytics data.
4. *(Optional)* Select Refresh to get the latest data.

<Callout kind="info" title="Note">

Metrics reflect data from the promotion's start date and are updated in near real time.
Data appears after a promotion is active and members engage with it.

</Callout>

5. (Optional) Select Download metrics to download the loyalty promotion analytics report.

## Viewing old loyalty promotions

You can view and manage live and upcoming loyalty promotions created on the old UI from the loyalty promotions listing page.

To view loyalty promotions created on the old UI, select Promotions via new UI from the drop-down menu.
