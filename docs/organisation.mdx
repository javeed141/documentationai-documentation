At Capillary, organizations (also referred to as 'orgs' in this document) are managed in three ways, based on the brand structure.

<Image src="https://files.readme.io/498097a4f9467c3f2844e368863b6d929702f8d24cfdcd7532cd18a0f69cd5d3-image.png" alt="" />

The table below lists the types of organizations, what it is, and the use cases that which they are suitable.

| Type of Org                                                                                                                                                                                                                 | What it is                                                                                                                                                          | Who is it suitable for?                                                                                                                                                             |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **[Independent / Standard Organization](https://docs.capillarytech.com/docs/organization-management#/)**                                   | A standalone organization that manages its own entities, customer data, loyalty programs, and engagement initiatives without any dependency on other organizations. | Businesses running independent loyalty programs where data sharing across entities is not required.                                                                                 |
| **[Organization Units (OUs) within a single organization](https://docs.capillarytech.com/docs/organisation-units-ou#/)** | A single organization set up for a conglomerate, where each brand is mapped to a dedicated OU.                                                                      | For conglomerates that want to manage loyalty operations across all their brands centrally.                                                                                         |
| **[Connected Organizations](https://docs.capillarytech.com/docs/connected-organizations#/)**                                                           | Each brand under the conglomerate is set up as an independent organization in the Capillary platform.                                                               | For conglomerates that want to manage loyalty operations across all their brands centrally, while also allowing each brand to have its own loyalty program and manage its own data. |
