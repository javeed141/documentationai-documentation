title: Create Zalo Template
---

You can use the Zalo notification service (ZNS) to send messages through Zalo, a popular communication channel in Vietnam.

## Prerequisite

An official Zalo account (OA) is required.

### Configuring the template

Configuration of the Zalo template can only be done through the Capillary Gateway support team.

### Approval of Zalo templates

The Zalo template needs to be approved and whitelisted by Zalo. This approval process is handled by the Capillary Gateway team/vendor. You can filter the Zalo templates in the Creatives by their status.

<Image src="https://files.readme.io/5a86a7428b0ae3eb9a9449e8b67c5e0092d62f3af2e2465df81b47ebea9cd6b4-image.png" />

### Using uploaded template

To use a uploaded template perform the following steps below:

1. From the **Engage+** homepage, navigate to the Creative section and select **Zalo** as the engagement channel.
2. On the Zalo listing page, you can view the uploaded templates. Based on your needs, navigate to a Zalo template and select **Overview**.
3. A modal appears where you can edit the Zalo template. However, you can update only the variable portion of the template.
4. You can replace variables with static text or [labels](https://docs.capillarytech.com/docs/labels). Each variable field `({#var#})` supports a maximum of 50 characters. If a label resolves to more than 50 characters, the message is not sent and is skipped for that user.
5. On the right side of the modal, select **Open Preview**. You are redirected to a tab where you can preview the Zalo content.

<Image src="https://files.readme.io/30f30fa11cccd36992cc93b89911cd1aff7e74450ad1b3db7311b40459fb52d0-gifyy_2.gif" alt="" />

6. (Optional) Select **[Preview and Test](https://docs.capillarytech.com/docs/test-and-preview-across-engagement-channels)** to preview the message content and test it with a set of [Test customers](https://docs.capillarytech.com/docs/configure-campaign-level-settings#test-customers) before sending it.

<Image src="https://files.readme.io/67480ff751731c5309e23c5c629218c01231658baf1b4658cc55a08401af559e-image.png" alt="" />

<Callout kind="info">

  **Note**

  Tag-filling functionality is not available in Preview and Test. You can only send a test message.

</Callout>
