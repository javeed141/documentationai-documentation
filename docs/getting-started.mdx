How to create your first dataflow in Connect+ — selecting a template, configuring blocks, and running the flow.

The recommended way to start with Connect+ is to explore the available **templates** — pre-built dataflow configurations for common use cases such as customer registration, transaction ingestion, data reconciliation, audience reload, and behavioral event ingestion.

Select a template that matches your use case, configure it for your org, and use **aiRA Coder** to generate transformation scripts or understand an existing configuration.
