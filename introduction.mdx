## Overview

Catapillar helps you organize and manage all your project documentation in one centralized space. You create, edit, and collaborate on docs effortlessly, keeping everything version-controlled and searchable. Whether you handle technical specs, user guides, or API references, Catapillar streamlines your workflow.

## Quick Start

Get up and running with Catapillar in minutes.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://app.catapillar.com` and create a free account using your email.
  </Step>
  <Step title="Create Project" icon="plus">
    Click the new project button and name it, like "My Docs Hub".
  </Step>
  <Step title="Add First Doc" icon="file-plus">
    
    ```markdown
    # Welcome to My Project
    
    This is your first document.
    
    ## Next Steps
    - Add more sections
    - Invite collaborators
    ```
    
  </Step>
  <Step title="Publish" icon="globe">
    Save and share the public link with your team.
  </Step>
</Steps>

## Why Choose Catapillar

You select Catapillar for its simplicity and scalability. Unlike traditional wikis, it integrates with your Git workflow and supports MDX for rich components like tabs and cards. Teams report 40% faster doc updates.

<Tabs>
  <Tab title="For Teams" icon="users">
    Unlimited collaborators, role-based access, and audit logs keep your docs secure.
  </Tab>
  <Tab title="For Solo Developers" icon="code">
    Markdown editor with previews, search, and export to PDF or HTML.
  </Tab>
</Tabs>
