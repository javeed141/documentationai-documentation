---
title: Project Not Visible
description: Fix missing project in dashboard.
---

## Symptoms

- A project you created does not appear in the catapillar dashboard.

## Possible Causes

- Project not created successfully
- Filter/search not matching project name

## Quick Fix

<Steps>
  <Step title="Check filters">
    Ensure you have not applied filters that hide your project.
  </Step>
  <Step title="Reload dashboard">
    Press Ctrl+R or click refresh icon in the dashboard.
  </Step>
  <Step title="Retry creation">
    Create the project again if it is still missing.
  </Step>
</Steps>
